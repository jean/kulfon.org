# Directory Structure

`kulfon new <project-name>` creates the following directory structure:

```
├── config.yml
├── public
└── website
    ├── components
    │   ├── layouts
    │   │   └── index.njk
    │   └── meta.njk
    ├── data.yml
    ├── images
    │   └── logo.png
    ├── javascripts
    │   └── main.js
    ├── pages
    │   ├── index.njk
    │   └── about.njk
    └── stylesheets
        └── styles.scss
```
